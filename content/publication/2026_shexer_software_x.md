abstract = "RDF shapes are effective tools for describing and validating RDF content. However, producing and maintaining them is a time-consuming task that typically requires domain experts with technical skills. In this paper, we present sheXer, an open-source Python library for the automatic extraction of RDF shapes from existing RDF instance data. sheXer supports a wide range of use cases by accepting multiple input formats (e.g., text files, SPARQL endpoints, compressed content), producing different types of outputs (SHACL, ShEx, UML visualizations, RDF-Config files, statistics), and enabling extractions tailored to specific goals."
